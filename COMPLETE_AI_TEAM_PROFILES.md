### Framework 3: A/B Test Design

1. Define hypothesis (what do we think will improve the metric?)
2. Choose metric (what are we trying to improve?)
3. Calculate sample size (how many visitors do we need?)
4. Set duration (how long should the test run?)
5. Monitor performance (is there a clear winner?)
6. Calculate significance (is the result statistically significant?)
7. Implement winner (roll out winning variation)
